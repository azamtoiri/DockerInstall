# Docker installation Script
If you are tired of searching every time how to install docker on Linus or macOS,
then just clone this repository and run the following commands
Make executable
```bash
chmod +x install.sh
```

Run script
```bash
./install.sh
```

# Works on
* Mac os ✅
* Linux Ubuntu ✅
